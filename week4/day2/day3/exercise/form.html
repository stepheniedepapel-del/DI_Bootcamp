Exercise 1: HTML Form (GET Method) GET

Create a form with name input, message textarea, and submit button. Data submits via GET.

📍 Where does the data appear?

Answer: With GET method, form data appears in the URL as query parameters.

Example: ?name=John&message=Hello+World

Visible in:

  • Browser address bar
  • Browser history
  • Server logs
  • Can be bookmarked

Exercise 2: HTML Form #2 (POST Method) POST

Same form structure, but using POST method. Check the Network Tab!

📍 Where does the data appear?

Answer: With POST method, form data appears in the HTTP Request Body, NOT in the URL.

💡 How to see it:
  1. Open DevTools (F12)
  2. Go to Network tab
  3. Submit the form
  4. Click the request in the list
  5. Look under Payload or Form Data

Characteristics:

  • Not visible in URL or browser history
  • Can send larger amounts of data
  • Better for sensitive information
  • Used for creating/updating resources

📋 Converted to JSON String

What happens to nested objects?

They are flattened into the JSON string recursively. JSON.stringify() converts the entire object hierarchy into a single string representation, maintaining the structure through nested braces.
